# Dtsivkovski Tap

## How do I install these formulae?

`brew install dtsivkovski/tap/<formula>`

Or `brew tap dtsivkovski/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dtsivkovski/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
